## Tooling and Build Downloads

- Rust toolchain (via `rustup`)
- `cargo` (part of Rust toolchain)
- Tauri build dependencies (via Cargo): `tauri`, `tauri-build`, `tauri-runtime`, `wry`, `tao`
- Release workflow tools (downloaded during CI):
- `cargo-deb` (Linux .deb packaging)
- `cargo-wix` (Windows .msi packaging)
- WiX Toolset (`wixtoolset` via Chocolatey on Windows CI)
- macOS `pkgbuild` (Apple system tool, not downloaded by repo)

## Runtime System Dependencies (not bundled)

- Windows: WebView2 Runtime (required by Tauri)
- macOS: WKWebView (system-provided)
- Linux: WebKitGTK and related GTK libs (system packages)

## Local Artifacts to Remove

- `target/` (Rust build outputs)
- `.env` (local secrets/config)
- `streaming.db` (SQLite data)

## Global Caches to Remove (if you want to fully clean the machine)

- `~/.cargo/registry` (downloaded crates)
- `~/.cargo/git` (git-sourced crates)
- `~/.rustup` (Rust toolchains)
